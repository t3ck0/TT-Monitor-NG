# TT-Monitor-NG
TT Monitor (Modernized) Modernisierter Fork der ursprünglichen App von NearbyExpress Tech. Dekompiliert und mit aktuellem SDK (Version 24+) neu kompiliert, um die Kompatibilität mit modernen Geräten bis Android 16 sicherzustellen. Reaktiviert das Bluetooth-Tracking für ältere Körperwaagen auf aktueller Hardware. Stabil, sicher und zukunftssicher.
